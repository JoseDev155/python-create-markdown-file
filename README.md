# Create Markdown File with Python

This repository contains a Python script that demonstrates how to create a Markdown file using the `pypandoc` library.

The script takes a string of Markdown content and converts it into a standalone Markdown file.

## ChatGPT example

The `main.py` file contains an example of how to use the `pypandoc` library to create a Markdown file.

It was generated with the help of ChatGPT, which provided the code and instructions for creating the Markdown file for an resume of my university studies (in Spanish).

## Usage

1. Create a virtual environment (optional but recommended):

```bash
python -m venv venv
source venv/bin/activate  # For Linux/Mac
venv\Scripts\activate     # For Windows
```

2. Install the required library or use the provided `requirements.txt`:

```bash
pip install pypandoc
pip install -r requirements.txt
```

3. Run the script:

```bash
python base_usage.py
```
